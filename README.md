# 株式会社八宝 コーポレートサイト

奈良市東向商店街を拠点に、飲食店・菓子店を展開する株式会社八宝のコーポレートサイトです。

## 構成

```
.
├── 八宝コーポレートサイト.html    # トップページ (バンドル)
├── 八宝の想い.html                  # 八宝の想い (理念)
├── 事業内容.html                    # 事業内容・店舗紹介
├── 採用情報.html                    # 採用情報トップ
├── 採用-grancha.html                # GRANCHA パティシエ求人
├── 採用-umasshu.html                # うまっしゅ 料理人求人
├── 採用-muginokura.html             # むぎの蔵 料理人求人
├── 採用-kamameshi.html              # 釜めし求人
├── 会社概要.html                    # 会社概要・お問い合わせ
├── プライバシーポリシー.html        # 個人情報保護方針
├── photos/                          # WebP 画像 (small/large)
├── fonts/                           # カスタムフォント (玉ねぎ楷書激 ほか)
└── videos/                          # 動画素材 (Git LFS / 外部ホスト)
```

## ローカルプレビュー

```bash
npx http-server . -p 8000 -c-1
# → http://localhost:8000/
```

## 主要店舗

- **GRANCHA** — 大和茶大福専門店 (奈良駅 東向商店街)
- **うまっしゅ** — 奈良の酒蔵全部呑み 居酒屋
- **むぎの蔵** — うどん・ごはん専門店
- **釜めし GRANCHA** — 釜めし専門店

## 注意

- `_素材/` (RAW 写真・動画) と `_バックアップ/` `_参考資料/` `動画編集/` は `.gitignore` で除外
- `videos/` 内 mp4 は容量が大きいため Git LFS / 外部 CDN で管理
- 編集後はブラウザで `npx http-server` でプレビュー確認すること
